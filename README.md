# Pastez - анонимный сервис хранения текста

Проект представляет собой backend-сервис для хранения и обмена пастами по ссылке, упрощённый аналог Pastebin.
Данный проект писался мною как первый тестовый проект, чтобы разобраться с структурой FastAPI-приложений, а также знакомству с другими библиотеками.

---

## Функционал

- Создание паст
- Получение пасты по уникальной slug-ссылке
- Автоматическая генерация короткой ссылки
- Подсчёт количества просмотров
- Анонимный доступ без регистрации и авторизации (в будущем планируется добавление регистрации)

---

## Технологии

- Python 3
- FastAPI
- SQLite
- SQLAlchemy
- Pydantic

## API

### Создать пасту

`POST /pastes`

**Body:**
```json
{
  "title": "Название",
  "content": "Текст пасты"
}
```

### Ответ

**Body:**
```json
{
  "slug": "Ab3x9k"
}
```

### Получить пасту

`GET /p/{slug}`

**Body:**
```json
{
  "title": "Название",
  "content": "Текст пасты",
  "views": 3
}
```

## Запуск проекта

# 1. Клонировать репозиторий

```
git clone url
cd pastez
```

# 2. Создать виртуальное окружение

```
python -m venv venv
```

# 3. Активировать окружение

```
.\venv\Scripts\Activate.ps1
```

# 4. Установить зависимости

```
pip install -r requirements.txt
```

# 5. Запуск

```
uvicorn app.main:app --reload
```
